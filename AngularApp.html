Angular To-Do-List

This project is a basic Angular application in which I learned to create components and manipualte their data. It has limited CRUD implementation with creates, updates, and deletes. (not to a database)

This application also introduces forms in Angular through the formBuilder. A single form takes input from the user, determines whether it is an update or a create, and creates a task from user input. Further, many OOP concepts exclusive to typescript were uesd in this project , ex. interfaces, classes.
